# Image Server - Frontend Template

This is a ready-to-use frontend template for the "Image Server" educational project. This project is a static website that you will bring to life with a Python backend.

## Getting Started

**To start working on the project, you should fork this repository.**

Forking is the standard workflow for using a template or contributing to a project on GitHub. It creates a personal copy of this repository under your own account, allowing you to track your changes, experiment freely, and manage your work professionally.

1.  Click the **Fork** button in the top-right corner of this page.
2.  Once you have your own fork, clone it to your local machine to begin development.
